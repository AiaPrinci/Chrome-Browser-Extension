# PDF Sentence Navigator

**PDF Sentence Navigator** არის Chrome Browser Extension, რომელიც PDF დოკუმენტში წინადადებებს შორის კლავიატურის საშუალებით გადაადგილების შესაძლებლობას იძლევა.

Extension შექმნილია **Chrome Extension Manifest V3-ისა** და **PDF.js-ის** გამოყენებით. PDF დოკუმენტი იხსნება Extension-ის საკუთარ viewer გვერდზე, სადაც აქტიური წინადადება ვიზუალურად ინიშნება.

## ძირითადი ფუნქციონალი

- `TAB` — გადასვლა მომდევნო წინადადებაზე;
- `SHIFT + TAB` — დაბრუნება წინა წინადადებაზე;
- აქტიური წინადადების ვიზუალური მონიშვნა;
- აქტიურ წინადადებამდე ავტომატური გადაადგილება;
- მიმდინარე წინადადებისა და გვერდის ნომრის ჩვენება;
- ონლაინ PDF ბმულების ავტომატურად გახსნა Extension-ის viewer-ში;
- ლოკალური PDF ფაილის არჩევა Extension-ის popup-იდან;
- PDF-ის ტექსტის დამუშავება მთლიანად ბრაუზერში.

## მუშაობის პრინციპი

Extension PDF დოკუმენტს ხსნის საკუთარ viewer გვერდზე.

PDF.js გამოიყენება PDF-ის ჩასატვირთად, გვერდების გამოსატანად და ტექსტის მისაღებად. მიღებული ტექსტი იყოფა წინადადებებად და თითოეული წინადადება უკავშირდება PDF-ის ტექსტურ ელემენტებს.

`TAB` ღილაკზე აქტიურდება მომდევნო წინადადება, ხოლო `SHIFT + TAB` კომბინაციაზე აქტიურდება წინა წინადადება.

ყოველი გადასვლისას:

- წინა წინადადებას მონიშვნა ეხსნება;
- ახალი წინადადება ვიზუალურად გამოიკვეთება;
- viewer ავტომატურად გადაადგილდება აქტიური წინადადების მდებარეობამდე;
- toolbar-ში ახლდება მიმდინარე წინადადებისა და გვერდის ნომერი.

## Extension-ის Chrome-ში ჩატვირთვა

1. ჩამოტვირთეთ ან დააკლონირეთ GitHub რეპოზიტორია.

```bash
git clone https://github.com/AiaPrinci/Chrome-Browser-Extension.git
```

2. გახსენით Google Chrome.

3. მისამართის ველში გადადით:

```text
chrome://extensions/
```

4. ჩართეთ **Developer mode**.

5. დააჭირეთ **Load unpacked**.

6. აირჩიეთ პროექტის ძირითადი საქაღალდე, რომელშიც მდებარეობს `manifest.json`.

7. Extension წარმატებით ჩაიტვირთება Chrome-ში.

## ფუნქციონალის შემოწმება

### ონლაინ PDF

1. გახსენით ნებისმიერი პირდაპირი PDF ბმული.

```text
https://example.com/document.pdf
```

2. PDF ავტომატურად გაიხსნება Extension-ის viewer-ში.

3. დაელოდეთ PDF-ის ჩატვირთვას.

4. დააჭირეთ `TAB` ღილაკს შემდეგ წინადადებაზე გადასასვლელად.

5. დააჭირეთ `SHIFT + TAB` კომბინაციას წინა წინადადებაზე დასაბრუნებლად.

6. გადაამოწმეთ, რომ:

- აქტიური წინადადება მონიშნულია;
- viewer ავტომატურად გადაადგილდება მონიშნულ წინადადებამდე;
- toolbar-ში ახლდება წინადადებისა და გვერდის ნომერი.

### ლოკალური PDF

1. დააჭირეთ Extension-ის icon-ს.

2. აირჩიეთ PDF ფაილი popup-იდან.

3. PDF გაიხსნება ახალ tab-ში.

4. გამოიყენეთ:

```text
TAB
```

შემდეგ წინადადებაზე გადასასვლელად.

და

```text
SHIFT + TAB
```

წინა წინადადებაზე დასაბრუნებლად.
